# Pasta App

Esta pasta contém toda a lógica da interface do usuário e as rotas principais da aplicação.

### Estrutura
- `page.tsx`: Página principal
- `layout.tsx`: Layout comum para todas as páginas
- `components/`: Componentes reutilizáveis
- `hooks/`: Hooks personalizados
- `context/`: Contextos globais
- `api/`: Rotas da API

### Objetivo
Organizar a interface do usuário em uma estrutura modular e fácil de manter.

### Notas importantes
- Cada nova página deve ser criada como um subdiretório dentro de `app/`
- Componentes devem seguir as convenções de nomenclatura no arquivo `types.ts`
