# Pasta Docker

Esta pasta contém configurações relacionadas ao containeramento do projeto.

### Estrutura
- `docker-compose.yml`: Configuração de serviços
- `Dockerfile`: Definição da imagem base

### Objetivo
Facilitar o deployment em ambiente controlado e consistente.

### Notas importantes
- Certifique-se que as imagens são atualizadas regularmente
- Use volumes persistentes para dados sensíveis
- Teste localmente antes de subir para produção
