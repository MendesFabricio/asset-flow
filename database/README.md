# Pasta Database

Esta pasta contém toda a lógica relacionada ao armazenamento de dados.

### Estrutura
- `models.py`: Definições de modelos do banco de dados
- `migrations/`: Histórico de migrações

### Objetivo
Prover um sistema de armazenamento consistente e escalável.

### Notas importantes
- Todas as alterações no schema devem ser acompanhadas por migrações
- Mantenha backups regulares dos dados
- Certifique-se da segurança das conexões
